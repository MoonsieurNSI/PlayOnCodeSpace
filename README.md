# PlayOnCodeSpace

> [!TIP]
> On s'amuse ici sur une VM Linux Debian mais
> ** the GitHub Codespaces infrastructure (which runs on Azure) block the ICMP protocol (the specific type of network packets that ping uses). They do this to prevent people from using their servers to launch DDoS attacks or scan networks.**
> donc on oublie `ping`, `traceroute`
> mais `curl` rules the world !
